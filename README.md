* Configuration
    - bundle install

* Install mongodb
    - brew install mongodb-community@6.0
    - brew install mongodb
    - brew services start mongodb-community
    - Add the path in your .zshrc or .bash_profile

* Run rake task to populate seeds data
    - rails db:seed:create_products

* Run server
    - rails s -p 3001
